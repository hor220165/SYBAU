## 📖 Über das Projekt

SYBAU ist eine innovative Fitness-Web-App, die Gamification-Elemente mit Workout-Tracking kombiniert. Nutzer können ihren Avatar leveln, Items sammeln, Quests abschließen und sich in globalen Leaderboards messen – während sie gleichzeitig ihre Fitness verbessern!

### 🎮 Warum SYBAU?

- **RPG-Mechaniken**: Levelsystem mit XP und Coins
- **Motivation**: Achievements, tägliche Quests und wöchentliche Challenges
- **Wettbewerb**: Globale Leaderboards
- **Belohnungen**: Item Shop mit Waffen, Rüstungen und Boosts
- **Personalisierung**: Individueller Avatar mit verschiedenen Ausrüstungsslots

---

## ✨ Features

### 🎯 Für Nutzer
- **Avatar-System**: Erstelle und personalisiere deinen Charakter
- **Workout-Tracking**: Logge verschiedene Workouts (Push-Ups, Squats, Laufen, etc.)
- **XP & Level-System**: Sammle Erfahrungspunkte und steige in Levels auf
- **Item Shop**: Kaufe Chests (Starter bis Legendary) und mächtige Items
- **Quest-System**: Tägliche und wöchentliche Herausforderungen
- **Achievements**: Über 100 freischaltbare Erfolge
- **Leaderboard**: Vergleiche dich mit anderen Spielern weltweit
- **Profile Management**: Statistiken, Fortschritt und Einstellungen
- **Boost-System**: Temporäre Verstärkungen für mehr XP/Coins

### 🛡️ Für Admins
- **User Management**: Verwalte Nutzer und Rollen
- **Challenge-Editor**: Erstelle und bearbeite Quests
- **Item-Editor**: Verwalte Shop-Items und Preise
- **Analytics**: Einsicht in Nutzerstatistiken

## 🛠️ Tech Stack

### Frontend
- **Framework**: Vue 3 (Composition API)
- **Language**: TypeScript
- **Build Tool**: Vite
- **Styling**: Custom CSS (Gradient Themes, Glassmorphism)
- **Routing**: Vue Router
- **State Management**: Composables + localStorage
- **HTTP Client**: Axios
- **Icons**: Lucide Icons / Custom SVG

### Backend
- **Framework**: ASP.NET Core 8.0
- **Language**: C#
- **Database**: SQLite (Entity Framework Core)
- **Authentication**: JWT Bearer Tokens
- **Architecture**: RESTful API
- **ORM**: Entity Framework Core
